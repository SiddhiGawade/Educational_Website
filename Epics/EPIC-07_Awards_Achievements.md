# EPIC-07: Awards & Achievements
**Priority:** P3 – Medium | **Sprint:** Week 4 | **Depends On:** EPIC-02

## Description
Display page showcasing student and institutional achievements with an image gallery. Admin manages content via admin portal (EPIC-02). This is the final feature with the lowest priority.

## Acceptance Criteria
- Awards page displays achievements with title, description, date, and images
- Image gallery with lightbox view for full-size photos
- Admin can add, edit, and remove awards content
- Page is responsive and visually appealing

## User Stories

| ID | Story | Points |
|----|-------|--------|
| US-26 | **View Awards Page** – As a visitor, I want to view achievements to feel inspired | 3 |
| US-27 | **Browse Image Gallery** – As a visitor, I want to browse award photos in a gallery | 3 |
| US-28 | **Admin Manages Awards** – As an admin, I want to manage awards content | 5 |
