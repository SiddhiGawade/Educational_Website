## Description

Create a dedicated page to showcase student and institutional achievements, awards, and recognitions. The page features an image gallery of award ceremonies and events along with descriptive text, motivating students to excel.

---

## Business Value

- Motivates students by recognizing excellence
- Builds credibility and trust with parents
- Showcases the platform's impact on student learning

---

## Acceptance Criteria

1. Awards page displays an image gallery with award photos
2. Each award/achievement has a title, description, and date
3. Images can be viewed in a lightbox/enlarged view
4. Admin can upload and manage awards content
5. Page is responsive and visually appealing

- **Assumption:** This is a display-only page; no student interaction required
