# EPIC-03: Interactive Quizzes
**Priority:** P1 – Critical | **Sprint:** Week 2 | **Depends On:** EPIC-01, EPIC-02

## Description
Daily quiz system where admin uploads quizzes, students attempt them after login, and results with leaderboard are revealed next day after teacher announcement. Requires student login (EPIC-01) and admin portal (EPIC-02) to be built first.

## Acceptance Criteria
- Admin can create and upload daily quizzes with MCQ questions
- Students can attempt quizzes after login (one attempt per quiz)
- Results remain hidden until admin declares them (next day)
- Leaderboard displays top performers after results are declared
- Students can view their quiz history and past scores

## User Stories

| ID | Story | Points |
|----|-------|--------|
| US-09 | **Admin Uploads Quiz** – As an admin, I want to create and upload a daily quiz for students | 8 |
| US-10 | **Student Attempts Quiz** – As a student, I want to attempt the daily quiz to test my knowledge | 8 |
| US-11 | **Declare Results** – As an admin, I want to declare quiz results so students can see their scores | 5 |
| US-12 | **View Leaderboard** – As a student, I want to see the leaderboard to know how I rank | 5 |
| US-13 | **View Quiz History** – As a student, I want to see my past quiz attempts and scores | 3 |
