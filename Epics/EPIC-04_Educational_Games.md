# EPIC-04: Educational Games
**Priority:** P1 – Critical | **Sprint:** Week 2–3 | **Depends On:** EPIC-01, EPIC-02

## Description
Interactive learning games with per-student score tracking and responsive interface. Requires student login (EPIC-01) for score tracking and admin portal (EPIC-02) for game management.

## Acceptance Criteria
- Students can browse and play interactive educational games
- Game scores are tracked and saved per student
- Games are responsive across desktop, tablet, and mobile
- Admin can upload and manage games through the admin portal

## User Stories

| ID | Story | Points |
|----|-------|--------|
| US-14 | **Play Educational Game** – As a student, I want to play learning games so I can learn in a fun way | 8 |
| US-15 | **Track Game Scores** – As a student, I want my scores saved so I can track my progress | 5 |
| US-16 | **Responsive Game Interface** – As a student, I want games to work on any device | 5 |
| US-17 | **Admin Manages Games** – As an admin, I want to upload and manage games from the portal | 5 |
