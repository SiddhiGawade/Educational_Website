# User Stories — EPIC-01: Homepage & Navigation

> **Phase:** Phase 1
> **Depends On:** None — this is the first epic to be developed.
> **Summary:** Build a clean, responsive homepage with clear navigation to all major sections of the website. The homepage serves as the main landing page for students, teachers, and visitors.

---

## US-01: Homepage Hero Section

> **As a** visitor, **I want to** see a welcoming hero section with a tagline and imagery, **so that** I immediately understand the purpose of the website.

**Acceptance Criteria:**
- Hero banner with educational theme image and tagline (e.g., "Welcome to the #1 Educational Hub")
- Target audience mentioned (Resources for 8th – 10th Grade Students)
- Inspirational quote displayed
- "Start Learning" call-to-action button
- Responsive on mobile, tablet, and desktop

---

## US-02: Main Navigation Bar

> **As a** visitor, **I want to** see a clear navigation bar, **so that** I can easily access all sections of the website.

**Acceptance Criteria:**
- Navigation links: Home, About, Career Guidance, Blog, Contact, Start Learning
- Navigation bar is sticky/fixed at the top
- Mobile-friendly hamburger menu on smaller screens
- Active page is highlighted in navigation

---

## US-03: About the Founder Section

> **As a** visitor, **I want to** see an "About the Founder" section on the homepage, **so that** I know the credentials and background of the educator behind the platform.

**Acceptance Criteria:**
- Founder photo, name (Mr. Pravin Vijay Panpatil), and brief bio displayed
- Key highlights: 24+ years of Academic Leadership, 10+ District & State Level Awards
- Section is visually clean and easy to read
- Links to the full About Me page

---

## US-04: Subject Categories Section

> **As a** visitor, **I want to** see subject category cards on the homepage, **so that** I can quickly navigate to content for English, Science, or General Knowledge.

**Acceptance Criteria:**
- Category cards displayed for: English, Science, General Knowledge
- Each card has a thumbnail image and subject name
- Clicking a card navigates to the respective subject's content page
- Cards are responsive and visually appealing

---

## US-05: Daily Updates Section

> **As a** visitor, **I want to** see a "Daily Updates" section on the homepage, **so that** I can quickly access the latest blog, career guidance updates, and quiz of the week.

**Acceptance Criteria:**
- Section shows latest updates with thumbnail, title, and date
- Categories shown: Latest Blog, Career Guidance, GK Quiz of the Week
- Each item links to the relevant external resource or page
- "View All" link to see all updates
